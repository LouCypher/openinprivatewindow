# Open in Private Window

Add more context menu items and options to open a URL in Firefox Private Window.

## Compatibility

Firefox 20.0 and newer.

## Installations

[Dev build](https://dl.dropbox.com/u/120550/extensions/openinprivatewindow.xpi).
